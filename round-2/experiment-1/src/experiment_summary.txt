# RVP Experiment Summary

## Experiment Completed Successfully

### Method Implemented
- Reading Vector Patterns (RVP) for near-duplicate short text detection
- Uses cognitive load indicators: word length, word frequency, POS tags
- DTW (Dynamic Time Warping) for sequence comparison

### Dataset
- Quora Question Pairs (QQP) 
- Train: 5000 examples (2500 duplicates)
- Val: 1000 examples (500 duplicates)
- Test: 4000 examples (620 duplicates)

### Results

| Method | Precision | Recall | F1-Score | ROC-AUC | Time/Pair (ms) |
|--------|-----------|--------|----------|----------|------------------|
| RVP    | 0.196    | 0.911  | 0.323    | 0.695    | 11.1             |
| Jaccard| 0.281    | 0.815  | 0.418    | 0.752    | 0.5              |
| MinHash| 0.294    | 0.777  | 0.426    | 0.757    | 11.6             |
| SBERT  | 0.461    | 0.653  | 0.541    | 0.873    | 67.8             |

### Key Findings
1. SBERT achieves best performance (F1=0.541)
2. RVP shows competitive ROC-AUC (0.695) but lower precision
3. Simple Jaccard baseline performs reasonably (F1=0.418)
4. RVP is 6x faster than SBERT

### Ablation Study (Top variants)
- length_only: F1=0.393
- length_frequency: F1=0.393
- full_rvp: F1=0.393
